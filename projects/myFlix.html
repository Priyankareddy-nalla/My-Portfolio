Project Description

myFlix is a client-side web application built with React that serves as a movie database and recommendation platform.It interacts seamlessly with an
existing server-side REST API to provide users with a comprehensive, dynamic, and user-friendly movie browsing experience. This project
demonstrates expertise in modern web development practices, focusing on building a Single Page Application(SPA) with a clean and responsive user interface.

Features

  • Single Page Application (SPA): Built using React, myFlix offers a smooth and seamless user experience with dynamic content updates, reducing the need for full-page reloads.
  • State Routing: Implemented state-based routing to facilitate navigation between different views, allowing users to share URLs and bookmark their favorite pages.
  • Movie Search and Filter: Includes a robust search feature that enables users to filter and find movies quickly based on titles and genres, enhancing user engagement and accessibility.
  • Responsive Design: Leveraged Bootstrap for styling and responsiveness, ensuring that the application provides an optimal viewing experience across various devices and screen sizes.
  • Modern JavaScript: Utilized ES2015+ syntax, taking advantage of features such as arrow functions, classes, and modules to write clean, efficient, and maintainable code.
  • Function Components: Used React function components to build the application, promoting better performance and readability with hooks for managing state and side effects.
  • Parcel Bundler: Employed Parcel as the build tool to simplify the development process, offering fast builds and hot module replacement for an efficient development workflow.
  • Online Hosting: The application is hosted online, making it accessible to users worldwide and showcasing the ability to deploy modern web applications in production environments.

Technologies used:

  • Frontend: React, JavaScript, Bootstrap
  • Build Tool: Parcel
  • API Integration: REST API for fetching and managing movie data
  • Hosting: Deployed on a cloud platform for public access

Challenges & Learnings

  • SPA Architecture: Developed a deep understanding of SPA architecture and its advantages in creating fast, responsive applications.
  • React & Modern JavaScript: Gained proficiency in React and modern JavaScript, particularly in using function components and hooks to manage application state and lifecycle events.
  • Responsive Design: Enhanced skills in designing responsive interfaces that adapt gracefully to different devices and screen sizes.
  • Build & Deployment: Learned to configure and use Parcel for building and deploying applications efficiently, understanding the importance of performance optimization in production environments.
