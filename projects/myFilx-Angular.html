Project Description

myFlix is a dynamic and interactive web application designed to provide users with an engaging
platform to explore and discover movies. The client-side of the application is built using Angular,
leveraging a RESTful API to interact with a robust server-side and database backend. This project
demonstrates my skills in front-end development, API integration, and creating a seamless user experience.

Features

    User Authentication and Authorization:
  • Secure login and registration functionalities using JSON Web Tokens (JWT) for user authentication.
    • Movie Catalog:
  • Display a comprehensive list of movies fetched from the server-side API.
    • User Authentication and Authorization:
  • Secure login and registration functionalities using JSON Web Tokens (JWT) for user authentication.
    • Movie Details:
  • Detailed view of each movie, including synopsis, genre, director, and other relevant information.
    • User Profile Management:
  • Users can view and update their profile information
    • Responsive Design:
  • Fully responsive UI that ensures a consistent experience across devices, including desktops, tablets, and smartphones.

    • Technologies used:

  • Database: MongoDB (accessed via the REST API)
  • Authentication: JSON Web Tokens (JWT)
  • Version Control: Git, GitHub
  • Deployment: Hosted on Git pages
